The basic:

Div, or blocks can have width and height. If it has width, remaining right space is taken by margin

block

Span, or inlines cannot have width and height

Blocks and Inlines can both have margin and padding, but inlines cannot have top/bottom margin

spanspan

Inline blocks behave like inlines, but can also have width&height and top/bottom margin

Inline Block

Floats:

Basic flow of display:block elements are top to bottom.

display: float; will allow us to take the element out of this flow, and position left or right.

Floats must have width

Next block element's background will fill the space and its contents will wrap around the floated element

float: none; will turn element go back into the flow

Horizontal Aligning

Inline items:

text-align: center. <-- this aligns content and inlines inside a block

text-align: center aligns content and inlines inside a block

Block items:

To center align, we can set left & right margin to auto.

Remember, a block with width takes all the margin on the right to create a block

when we set left & right margin to 'auto' it will center the block in its parent.

block

We can float left/right to left/right align block elements

We can also use Flexbox to align block elements however we want
