Wrapping items: flex-wrap

flex-wrap: wrap

flex-wrap makes elements behaves like floated items, except with more control.

Container Direction: flex-direction: row | row-reverse | column | column-reverse

This is whether the flow is horizontal or vertical.

Notice how the column is not centered even with justify-content: center

This is because when flex-direction is row, the direction is flipped side ways too

So: normal/row, justify content is <--->

but when flex-direction: column, justify content is top-to-bottom

thus to horizontally center a column, we must align-item: center

Override Item order: order: (number)

So far we changed everything through the container, but this goes on individual flex items

by default this is 0.

Likewise.. we can also:

Override Item align: align-self: flex-start | flex-end | center | stretch | baseline

to override each item's alignment


Flex box

display: flex

As we know this makes the child items a flex item.

flex: (number)

this will determine the weight of the width of each item

All items widths will be divided equally if they all have the same flex number equally.

Eg. Item with flex 2 will grow twice as fast flex: 1.
