Positioning -- static, relative, absolute & fixed

Most of the time, we want everything to be static. Use positioning for fine tuning


Relative Position

This is moving an element relative to where their normal position would be

everything else renders as if the relative item is in static position

we can use top/bottom/left/right (number) to offset

Absolute

Absolute offsets from the nearest parent with relative position till window

bottom: 0 on window will vary as window height changes

Absolute position removes the element from the flow as if it doesnt exist.

Absolute position is more useful when paired with relative position.

Fixed

Fixed is similar to absolute where element is removed from normal flow but fixed is always relative to window

Key difference is that fixed doesnt scroll with the rest of the page

Mainly good for top navs that always stay.
